# Website Edukasi Kesehatan Jasmani & Rohani

Website edukasi tentang pentingnya keseimbangan antara kesehatan jasmani dan rohani untuk hidup yang lebih berkualitas dan bermakna.

## Fitur
- Desain responsif yang menarik
- Animasi dan transisi yang halus
- Layout grid untuk konten kesehatan
- Tip dan saran praktis
- Optimasi untuk semua perangkat
